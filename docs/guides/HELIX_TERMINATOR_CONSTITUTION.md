overridden below with an explicit `Override §X.Y` section.

no overrides are in force. The parent repository is a Go skeleton; when
genuinely universal rules emerge they will be contributed upstream to the
constitution submodule via pull request (never committed directly to a
tagged release), per the Constitution's promotion policy. Project-specific
configuration stays in this file and in the parent `CLAUDE.md` /
